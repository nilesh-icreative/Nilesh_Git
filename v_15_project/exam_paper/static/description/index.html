Exam Paper

Confirm

  • Users should only be able to select sale orders which are in draft or in quotation sent.
  • On click of Proceed Operation button, it will confirm the selected sale order.

Cancel

  • Users should only be able to select sale orders which are in draft, quotation sent, sale.
  • On click of Proceed Operation button, it will cancel the selected sale order.

Merge

  • Users should only be able to select sale orders which are in draft or in quotation sent and which customer is as per selected customer.
  • On click of Proceed Operation button, it will merge the selected sale order into new sale order and cancel selected sale orders.

Prerequisites before Installation

No prerequisites are required before the installation.

Configurations

When user selects Responsible in batch then allow user to select only from those sales order whose salesperson is same as selected batch responsible user.

Once the user clicks on Proceed Operation, it will change state to Done and pass the Operation date in the selected sale orders.

When you click on the “Operation Type” you will be asked to choose from Three different options before create orders.

As shown in the below image you can:

  • Confirm
  • Cancel
  • Merge
